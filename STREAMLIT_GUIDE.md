# 🚀 Streamlit 웹 앱 실행 가이드

## 빠른 시작

```bash
# 1. 패키지 설치 (처음 한 번만)
pip install -r requirements.txt

# 2. Streamlit 앱 실행
streamlit run app.py
```

브라우저가 자동으로 열립니다! 🎉

---

## 📱 기능 소개

### 1. 📝 문제 풀기

- **문제 유형 선택**: 코드 문제 또는 이론 문제 선택
- **문제 생성**: AI가 Few-shot 기반으로 새로운 문제 생성
- **답변 제출**: 답을 입력하고 정답 확인
- **즉시 피드백**: 정답/오답 여부와 해설 제공

**사용 방법:**
1. 사이드바에서 "📝 문제 풀기" 선택
2. 문제 유형 선택 (코드/이론)
3. "🎲 문제 생성" 버튼 클릭
4. 생성된 문제 읽고 답변 입력
5. "제출" 버튼 클릭
6. 결과 확인!

---

### 2. 🔁 복습하기

- **틀린 문제 목록**: 이전에 틀린 문제들 확인
- **벡터 DB 저장**: 자동으로 틀린 문제 저장
- **복습 학습**: 취약한 부분 집중 학습

**사용 방법:**
1. 사이드바에서 "🔁 복습하기" 선택
2. "📚 복습 문제 불러오기" 버튼 클릭
3. 틀렸던 문제 확인
4. 정답과 비교하며 복습

---

### 3. 📊 통계

- **전체 통계**: 총 문제 수, 정답률 확인
- **유형별 통계**: 코드/이론 문제별 성적
- **학습 진도**: 시각적으로 진행 상황 확인

**표시 정보:**
- 총 문제 수
- 정답/오답 개수
- 정답률 (%)
- 코드 문제 정답률
- 이론 문제 정답률

## 🔧 커스터마이징

```bash

```

## 🐛 트러블슈팅

### "Module not found" 오류
```bash
pip install -r requirements.txt --upgrade
```

### 포트 이미 사용 중
```bash
# 다른 포트로 실행
streamlit run app.py --server.port 8502
```

### 벡터 DB 오류
```bash
# 벡터 DB 초기화
rm -rf vector_store/
streamlit run app.py
```

### OpenAI API 오류
- `.env` 파일에 API 키 확인
- API 사용량 확인
- 인터넷 연결 확인
